# Random Quote Generator

## Автор
Алина Сергеева

## Описание
Это приложение "Random Quote Generator" позволяет пользователям генерировать случайные цитаты из предопределенного списка. Пользователи могут также просматривать историю сгенерированных цитат, фильтровать их по автору и теме, а также сохранять и загружать историю в формате JSON.

## Установка и запуск
1. Убедитесь, что у вас установлен Python (версия 3.xх)
2. после того, как вы убедились, что у вас установлен Python откройте файл main.py
